# Pony_Cursor

给 Cursor 的代码。

## 目录

| 目录 | 说明 |
|------|------|
| `desktop-pet/` | 桌面宠物（Python + PyInstaller） |
| `opencode/` | OpenCode CLI 相关 |
| `game/` | Python 小游戏 |

## 本地运行

```bash
# 桌面宠物
cd desktop-pet
pip install -r requirements.txt
python pet.py

# 游戏
cd game
pip install -r requirements.txt
python main.py
```
