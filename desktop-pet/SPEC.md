# Desktop Pet - 桌面宠物规范文档

## 1. 项目概述

- **项目名称**: Desktop Pet (桌面宠物)
- **项目类型**: 桌面应用程序
- **核心功能**: 一个悬浮在桌面上的可爱卡通宠物，具有多种动画状态和交互功能
- **目标用户**: 想要桌面伴侣的用户

## 2. UI/UX 规范

### 2.1 窗口设计

- **窗口类型**: 无边框透明窗口
- **窗口大小**: 128x128 像素
- **窗口特性**: 
  - 置顶显示 (always on top)
  - 可拖拽移动
  - 无任务栏图标 (toolwindow)
  - 透明背景

### 2.2 视觉设计

- **宠物设计**: 简化的卡通小动物 (小黄鸡)
- **颜色方案**:
  - 主体: 黄色 (#FFD93D)
  - 嘴巴: 橙色 (#FF8C00)
  - 眼睛: 黑色 (#000000)
  - 腮红: 粉色 (#FFB6C1)
- **动画帧率**: 10 FPS
- **动画类型**:
  - Idle (待机): 轻轻摇摆
  - Walking (行走): 左右移动
  - Sleeping (睡觉): 闭眼打盹
  - Happy (开心): 跳跃

### 2.3 交互设计

- **左键拖拽**: 移动宠物位置
- **左键点击**: 触发开心动画 + 提示信息
- **右键菜单**: 
  - 喂食 (随机一种食物表情)
  - 睡觉/唤醒
  - 关于
  - 退出

## 3. 功能规范

### 3.1 核心功能

1. **宠物显示**: 透明窗口显示卡通宠物
2. **位置拖拽**: 鼠标拖动改变宠物位置
3. **自动移动**: 随机时间后自动小范围移动
4. **状态切换**: 点击/喂食改变宠物状态
5. **右键菜单**: 提供操作选项
6. **边缘吸附**: 移动到屏幕边缘时自动隐藏部分身体

### 3.2 动画状态机

```
[Idle] --点击--> [Happy] --2秒--> [Idle]
[Idle] --随机--> [Walking] --2秒--> [Idle]
[Idle] --右键--> [Sleeping] --点击/右键--> [Idle]
```

### 3.3 系统托盘

- 显示宠物图标
- 右键菜单: 显示/隐藏宠物、退出

## 4. 验收标准

- [ ] 宠物窗口透明且置顶
- [ ] 可以用鼠标拖拽移动
- [ ] 点击宠物会切换到开心状态
- [ ] 右键菜单功能正常
- [ ] 动画流畅无卡顿
- [ ] 程序可以正常退出
